# Four Seasons Markdown Editor 插件立项卡

## 一、来源

来自私有 Markdown 编辑器本机原型。

本机原型路径只保留在私有工作记录中，不进入公开版目录。
公开版只保留通用插件结构、可运行编辑器、配置说明和不含正文的演示内容。

## 二、核心能力

- 在 Codex 内置浏览器中打开本地 Markdown 编辑器。
- 支持搜索、主题切换、Markdown 简易格式工具。
- 支持 `?path=` 直达某个 `.md` 文件。
- 保存前显示 diff。
- 保存前生成备份。
- 历史版本回溯并载入为草稿。
- 基础文本检查。
- 多语言界面与主题/字体显示设置。

## 三、公开版边界

- 工作区通过启动参数、环境变量或 `editor/config.json` 配置。
- 不写死私人路径。
- 不提交 runtime/backups/activity/config。
- 示例 Markdown 不含正文和设定。
- README 面向普通 Codex 用户。
- 说明 Codex 当前无法接管原生文件链接，只能提供 editor URL。

## 四、官方反馈方向

建议向 Codex 官方反馈：

> 希望插件可以声明文件链接处理器，让 Markdown 编辑器类插件接管 `.md` 文件链接的打开方式。

英文标题建议：

