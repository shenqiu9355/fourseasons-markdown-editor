# Changelog

## 0.2.22

- Synced the public editor UI with the current working layout.
- Added line ruler, active-line highlight, cursor accent color controls, and bottom toolbar layout.
- Added Markdown file rename and workspace folder creation.
- Added a public `version.json` and a non-blocking update reminder.
- Kept public source free of private content, private paths, runtime state, backups, activity logs, and map artwork.

## 0.1.0

- Added the runnable Four Seasons Markdown Editor local web app.
- Added configurable workspace loading through `editor/config.json`, launch arguments, or environment variables.
- Added Markdown editing, preview, diff, save, history, search, text check, and display controls.
- Expanded editor font stacks for Chinese, English, Japanese, Korean, serif, sans-serif, and monospace reading.
- Added Ctrl/Command + mouse wheel editor font-size zoom.
- Added more Markdown formatting tools: numbered list, inline code, strikethrough, link, and horizontal rule.
- Expanded interface languages to English, Simplified Chinese, Traditional Chinese, French, German, Spanish, Portuguese, Italian, Dutch, Polish, Russian, Turkish, Arabic, Hindi, Indonesian, Vietnamese, Thai, Korean, and Japanese.
- Added one-time local acknowledgement for Simplified Chinese, Traditional Chinese, and Japanese language packs.
- Added Windows and macOS/Linux launch scripts.
- Kept runtime state, backups, logs, local config, private content, and private paths out of the public tree.
